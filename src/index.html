4 Second Word Game
Each round you will be given 3 letters.
You will have 4 seconds to enter a word which begins with the first letter and contains the other 2 letters in order.

For example for the letters RGN you could answer RINGING. You could not answer RING because that doesn't have a G before an N.

Each game consists of 10 rounds each lasting 4 seconds.

Good luck!
